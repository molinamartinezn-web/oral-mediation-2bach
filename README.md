# Oral Mediation Practice – 2nd Bachillerato

## Objetivo didáctico

Este recurso permite practicar la mediación oral en inglés a partir de un cartel breve en español sobre hábitos saludables. El alumnado debe identificar las ideas principales, valorar distintos modelos de respuesta y reconocer qué características hacen que una mediación sea completa, clara y natural.

## Nivel y grupo destinatario

Recurso dirigido a alumnado de **2.º de Bachillerato**, especialmente para la preparación de tareas de mediación oral en la materia de Inglés.

## Instrucciones de uso en clase

1. El alumnado lee la situación comunicativa y observa el cartel en español.
2. Revisa los pasos de preparación y las expresiones útiles para organizar una mediación oral.
3. Prepara oralmente una explicación breve del cartel en inglés.
4. Selecciona la mejor respuesta entre las tres opciones y pulsa **“Check my answer”**.
5. Lee el feedback, identifica por qué la respuesta es correcta o incorrecta y utiliza **“Try again”** para repetir la actividad si es necesario.

## Cómo probar el recurso en local

1. Descarga y descomprime la carpeta del proyecto.
2. Comprueba que `index.html`, `css/style.css` y `js/script.js` mantienen la misma estructura de carpetas.
3. Abre `index.html` con un navegador web como Chrome, Edge o Firefox.
4. Selecciona cada una de las respuestas y pulsa **“Check my answer”** para comprobar el feedback.
5. Opcionalmente, abre `F12 > Console` para verificar que no aparecen errores relacionados con la interacción.

## Valor didáctico

El recurso no se limita a indicar si una opción es correcta o incorrecta. Cuando el alumnado falla, resalta la opción elegida, muestra la respuesta adecuada y explica qué información falta o por qué la formulación es demasiado imprecisa. De este modo, el feedback orienta la mejora de futuras producciones orales.
